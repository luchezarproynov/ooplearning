
Simple data types:
    1. Numeric types:
    - int
    - float
    - Linux timestamp. Integer seconds which starts from 01-01-1970 00:00:00.
    
    2. String types:
    - string
    
    3. Boolean types:
    - bool
    
    4. Binary types:
    - any file
    
    5. Object types:
    - instances of any class or interface
    - DateTime class defines any date-time operations. It doesn't use timestamp.
